Experience

Arduino-Based Coconut Grater and Milk Extractor (Thesis Project)

  • Designed an automated machine that can both grate coconut and extract its milk using servo motors, relays, and sensors.
  • The system uses programmed logic and delays to control mechanical arms for coconut movement and pressing.
  • It was built to support small-scale vendors needing efficient coconut processing. But this project need more innovation

RFID Door Lock System with Solenoid

  • Created a basic access control system using an RFID module, a solenoid lock, and an LCD display powered by Arduino.
  • The system allows authorized tag access and displays status feedback on screen.
  • It served as a learning platform for embedded security systems and user interface integration.

Ultrasonic Radar System

  • Developed a simple radar-like system using an ultrasonic sensor and servo motor to detect object distances.
  • The data was displayed visually on a serial monitor to simulate obstacle detection.
  • This project enhanced my understanding of sensor readings, motion control, and real-time data output.

AI Voice Assistant (Python + Arduino)

  • Built a basic voice-controlled assistant using Python and ChatGPT API to process voice commands.
  • The assistant could control Arduino-connected devices like LEDs based on verbal instructions.
  • This project demonstrated simple AI integration with hardware through serial communication.

Piso WiFi and Car Wash Vending Machine

  • It introduced me to practical hardware deployment, coin-acceptance modules, and self-service business tech.
